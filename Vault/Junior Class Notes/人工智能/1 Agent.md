## Agent
> **agent is an entity that perceives and acts**
> agent是一个能感知并行动的实体

- Agent可以被视为：通过传感器感知环境，并通过执行器对该环境产生作用
- Agent：人类Agent 机器人Agent  软件Agent
- 感知：表示任何时刻Agent的感知输入
- 感知序列：该Agent所收到的所有输入数据的完整历史
- - 从数学角度：Agent函数描述了Agent的行为。它将任何一个给定的感知序列映射为一个实体。

**Agent的分类：**
- 简单反射Agent 
- 基于模型的反射Agent 
- 基于目标的Agent 
- 基于效用的Agent 
- 学习Agent
